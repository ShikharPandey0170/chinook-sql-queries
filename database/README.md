# Database

This folder contains the Chinook sample database used throughout this project.

## Files

- `chinook_mysql.sql` – SQL script used to create the Chinook database, including tables, relationships, and sample data.

## How to Import

1. Open MySQL Workbench.
2. Create a new schema (e.g., `chinook`).
3. Open `chinook_mysql.sql`.
4. Select the `chinook` schema.
5. Execute the script to create and populate the database.
